1. Install OpenVPN Access Server
2. Stop openvpnas service: systemctl stop openvpnas
3. Open file /usr/local/openvpn_as/lib/python/pyovpn-2.0-py[YOUR_PYTHON_VERSION].egg with archiver 7-zip or similar
4. Replace /usr/local/openvpn_as/lib/python/pyovpn-2.0-py[YOUR_PYTHON_VERSION].egg -> pyovpn/lic/info.pyc 
	 with cracked one from ./pyovpn-2.0-py[YOUR_PYTHON_VERSION].egg/info.pyc
5. Start openvpnas service: systemctl start openvpnas
6. Open admin panel at http://[YOUR_HOST]. Go to Configuration -> Activation tab. 
7. Expand Offline Activation and download Machine File machine-id.txt
8. Put machine-id.txt alongside with openvpn-as-kg.exe
9. Run keygen openvpn-as-kg.exe
10. Upload generated license file from C:/Users/YOUR_ACCOUNT/Desktop/XXXX-XXXX-XXXX-XXXX.lic

Enjoy!
